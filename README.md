# Photo-booth

This is the code for a Raspberry Pi photo-booth in python.

## Hardware
- Raspberry Pi 2 Model B
- Pi Camera
- Jump wires and buttons
- T-Cobbler for Raspberry Pi 2
- SD-Card
- USB-Hub
- Wlan-adapter
- HDMI Display
- HDMI cable


## Software
- Raspbian
- Python
- picamera
- gpio
